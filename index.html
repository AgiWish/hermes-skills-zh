日报 / 站会报告

/daily-standup-zh

输入今天做的事，自动生成标准三段式日报（昨日完成 / 今日计划 / 风险阻塞）

效率 日报 站会
▼
适用场景
  • 「帮我写日报」「生成今天的站会内容」
  • 「今天完成了 X，明天要做 Y」
  • 群里有人 @你 发日报
快速用法
/daily-standup-zh [今日完成的工作] # 可选参数 --tomorrow=[明日计划] # 明确指定明日计划 --block=[阻塞事项] # 有需要同步的阻塞 --style=简洁 # 纯文字，适合微信群 --style=标准 # 带 emoji 标题（默认）
支持平台

请假申请

/leave-request-zh

一句话描述请假原因，自动生成正式请假申请，支持正式 / 微信消息版

沟通 请假 申请
▼
适用场景
  • 「帮我写请假申请」「我要请假，帮我说一下」

▼
